# MangaBoard - マンガ・ネーム制作支援ツール

MangaBoardは、マンガのネームと絵コンテを同時に作成できるアプリです。
コマごとに画像やセリフを分割して管理することで、入れ替えや構成を簡単に行えるようになっています。
外部ペイントソフト（CLIP STUDIO PAINT、Photoshop等）で画像を編集でき、画像とテキストを別々に出力可能です。

## ✨ 主な機能

### 1. 三つの表示モード
作業の目的に合わせて、最適な表示形式を選択できます。

- **Overviewモード（ネーム表示）**
  - 全ページのコマ割りを一覧表示します。セリフや画像の入れ替え、配置、コマ数の変更をクリックとドラッグだけで自由に行えます。
- **List Overviewモード（絵コンテ表示）**
  - コマを縦に並べて絵コンテ形式で表示します。コマ割りを後回しで考えられるので、ストーリーだけに集中できます。
- **Split Viewモード（ネーム、絵コンテ同時表示）**
  - OverviewとList Overviewを左右に同時表示します。全体の流れを見ながら細部を調整する、最も効率的な編集モードです。

### 2. 強力な外部エディタ連携（ホットリロード）
- ツール内の画像を編集ボタンからCLIP STUDIO PAINT、Photoshop等の外部エディタで直接開けます。
- **外部エディタで保存すると、アプリ側の表示が即座に更新されます。** アプリを操作することなく、描画と確認を繰り返すことが可能です。

### 3. 直感的なアセット操作
- **画像ドラッグ＆ドロップ**: 画像ファイルをウィンドウに放り込むだけで、パネルとして即座に配置されます。
- 配置した画像は自由に入れ替えられます。
- Shift＋スクロールで画像の拡大、Shift＋ドラッグで画像位置を移動できます。
- **セリフ管理**: セリフをアセットとして管理し、自由な位置に配置・移動できます。
- 文字サイズ、縦書き横書きも指定できます。
- **コマ管理**: セリフと画像のセットをコマとして管理し、自由な位置、ページに配置・移動できます。

### 4. PSD・画像サポート
- 各コマはPSDファイルで作成されます。jpg,png,bmpを読み込むことも可能です。

### 5. コマ割り変更
- コマをダブルクリックすることで左右分割、Ctrl＋ダブルクリックで上下分割できます。
- **コマの並び順がネームと絵コンテで同期するのは初期の６コマ配置のみです。コマ数を変更した後は手動で変更が必要になります。**

## 🚀 はじめかた

### 動作環境
- **OS**: Windows 10 / 11
- **依存環境**: Microsoft Edge WebView2 Runtime
  - 通常は標準搭載されています。起動しない場合は別途インストールが必要です。

### インストール・実行
1. [Releases](https://github.com/chchannel/MangaBoard/releases) から最新の `tauri-appmangaboard-proto.exe` または `MangaBoard_Setup.exe` をダウンロードします。
2. 実行ファイルをダブルクリックして起動します。
   - ※未署名のアプリのため、Windowsによって「保護」される場合があります。「詳細情報」をクリックし、「実行」を選択してください。
